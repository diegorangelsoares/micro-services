# micro-services

Projeto de reciclagem de conhecimento contento:
  - Java 11
  - Springboot
  - Springdata
  - Springcloud
  - Postgres
  - Zipkin
  - Rabbitmq
  - Eureka
  - ApiGateway
  - Docker / Docker-compose
  - Swagger
  - Feign
  - Actuator


Comando docker para subir o ZipKin: 
docker run -p 9411:9411 openzipkin/zipkin:2.23.2
